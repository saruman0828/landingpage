# Image Briefs

Create image briefs as Markdown files under `case-studies/construction-ai-case-assets/`.

## Rules

- Include target filenames.
- Use 16:9 unless another layout is explicitly needed.
- Avoid readable text, logos, real company names, and product UI.
- Show realistic Japanese SMB work scenes.
- Prefer visual stories: before/after, material -> draft -> human approval, manager regained focus time.
- Avoid generic robot, brain, glowing network, or purely abstract AI imagery.

## Useful Scenes

- Field worker sending a mobile instruction while on site.
- Photos and reports being organized into a draft.
- Store manager reviewing a shift candidate.
- Inventory exceptions highlighted without readable UI text.
- Factory owner pressing a start button for estimate preprocessing.
- Staff asking an in-store AI manual on a tablet.
- Sales rep reviewing customer research before a meeting.
- Owner seeing a prepared draft at the end of the day.
