# 社長の右腕AI LP

「社長の右腕AIを参謀に！」LPをメインとして管理するNext.jsプロジェクトです。AIマスター2日間集中キャンプLPとA/B/C案は、内容を混ぜず独立した別ページとして残しています。

## ローカル確認

```bash
pnpm install
pnpm dev
```

確認URL:

- 社長の右腕AI・参謀LP: http://localhost:3000
- 旧URLからの転送: http://localhost:3000/right-hand-ai
- AIマスター2日間集中キャンプ: http://localhost:3000/two-days
- A 現場改善型: http://localhost:3000/variant-a
- B案の旧URLからの転送: http://localhost:3000/variant-b
- C 経営者解放・仕組み継承型: http://localhost:3000/variant-c

## 統合方針

社長の右腕AI LPをトップページにしています。旧URLの `/right-hand-ai` は重複ページを作らず、トップページへ転送します。

- 社長の右腕AI・参謀LP: `/`
- 旧URL: `/right-hand-ai` から `/` へ転送
- AIマスター2日間集中キャンプ: `/two-days`
- B案の旧URL: `/variant-b` から `/two-days` へ転送
- A/C案: `/variant-a`, `/variant-c`
- 社長の右腕AI LPの静的素材は `public/main` に集約

## A/B/Cデザイン案

### A：現場改善型

URL: `/variant-a`

会社に持ち帰っている確認を、現場で即断できる基準に変える訴求です。建設業、不動産業、地域サービス業のように、社長が現場判断の最終確認者になりやすい会社に刺さります。

経営者に刺さる理由:

- 「またこの確認か」という日常の負担を、現場で即断できる余地として見せている
- 採用しても判断基準が共有されていなければ社長の雑務は減らない、という経営者の実感に近い
- AIツール導入ではなく、社長の判断基準を現場で使える形に変えるため、高額な実装支援として伝わる

### B：AIマスター2日間集中キャンプ

URL: `/two-days`

旧URL `/variant-b` は、重複ページを作らず `/two-days` へ転送します。

経営者・幹部が、生成AIを触って終わらず、自社業務で使える実務プロンプトと社内展開ルールまで作る訴求です。ChatGPTなどを使い、会議、営業、見積、採用、資料作成、教育、問い合わせ対応に置き換える2日間の集中講座です。

経営者に刺さる理由:

- AIを学びたいが、経営や幹部業務のどこに使うべきか決まっていない会社に合う
- 会議、営業、見積、採用、資料、教育、問い合わせ対応という日々の幹部業務に直結している
- 操作説明で終わらず、実務プロンプト、情報管理ルール、30日間の実践計画まで作るため、高額講座として伝わる

### C：経営者解放仕組み継承型

URL: `/variant-c`

社長の「またか」という雑務を減らしながら、社長の勘と経験を社員が使える会社の型に変える訴求です。属人化、後継者、教育、品質のばらつきが課題の会社に向きます。

経営者に刺さる理由:

- 「社長に聞けば分かる」状態の危うさを突いている
- 作業だけでなく判断基準を残すため、単なるマニュアル作成より価値が高く伝わる
- 会社に残る資産として見えるため、高額でも納得されやすい

## 共通設計

- CTA文言: `AIマスター集中キャンプの相談を予約する`
- 固定CTA: iPhone表示で画面下に表示
- 計測イベント:
  - `page_view`
  - `cta_click`
  - `nav_click`
  - `form_start`
  - `form_submit_attempt`
  - `form_submit_success`
  - `form_submit_error`
  - `calendar_open`
  - `calendar_booked`（`?booked=1` で発火）
- 禁止表現:
  - 紫中心のAI風デザイン
  - ロボット、宇宙、ネオン、未来都市
  - 架空実績、架空口コミ、架空ロゴ
  - 根拠のない売上倍率や過剰な削減率

## 実績・根拠の扱い

掲載する数字は、現時点で使える業務単位の変化に限定しています。

- 不動産会社: 毎朝1〜2時間かかっていたリサーチ、メール確認、返信業務の負担を大きく削減
- 古美術・買取業: 毎日約1時間の相場チェック業務を大幅に短縮
- 建設業: 仮見積、工程確認、法令確認などの判断補助を整理

会社名、人物名、ロゴ、口コミは、公開許可がある場合のみ掲載します。
