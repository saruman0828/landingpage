- 新しい用途は120ユースケースと比較し、異なる作業だけ追加する。
- 新しい仕組みは24の基礎パターンと照合し、派生例か新型かを記録する。

- LPや導線の案はLP営業活用へ置き、調査事実と実装計画を混ぜない。
