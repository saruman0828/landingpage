# 公開記事と出典

## 役割

case-studies配下の匿名ケースと、根拠となる公開情報を結び付ける。公開HTMLへ企業名や出典URLを出さない場合も、このフォルダで根拠を追跡できる。

## 資料

- 00_公開記事出典索引.md: 公開30記事の管理状態と移行先
- 05_記事化した匿名ケース対応表.md: 初期記事の対応
- 08_追加20件_匿名ケース対応表.md: 追加20記事の対応

## 記録する項目

1. 公開記事のパス
2. 匿名ケースの業種と止まっていた作業
3. 主な公開出典
4. 記事へ取り込んだ本質

新規記事を追加した場合は、まず00_公開記事出典索引.mdへ追記し、記事slugと同名の出典台帳を作る。企業名、製品名、数値を公開記事へ転記する前に、出典の公開日と原文を読む。
