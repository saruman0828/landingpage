# LP・営業活用

## 役割

調査結果を、LP、事例一覧、個別記事、営業説明へ展開するための資料。調査事実とサイト実装を混同しない。

| 資料 | 用途 |
|---|---|
| 02_LP活用メモ.md | 初期の訴求案、記事候補、営業説明 |
| 14_landingpage導入改善案.md | サイト全体の改善バックログ |
| 15_改善事例_全体改善計画.md | 事例領域の詳細設計と実装工程 |

14を上位計画、15を事例ページの詳細設計、02を表現と営業質問の素材集として扱う。資料に進捗状態は記録されていないため、実施済み・未実施は現行HTMLを読んで判断する。

公開企業名の扱いは、トップページの第三者事例枠では出典とともに掲載できるが、匿名ケースの一覧カードでは出さない。個別文案と人の担当範囲は、独立した免責表ではなく仕事の流れへ入れる。
