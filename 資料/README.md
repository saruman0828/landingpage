# 資料フォルダ

作成日: 2026-06-20

新規訪問者が見て、納得し、連絡を取りたくなるWeb集客用の再設計資料です。

主ターゲットは中小企業の社長です。ただし、社長本人だけに限定しすぎず、役員、後継者、店長、事業責任者、管理部門長、社長から業務改善を任されている社員も問い合わせできる表現にします。

前回の「AI調査資料」ではなく、今回は以下に絞っています。

- Web集客に成功しているサービスサイトの見せ方
- 新規客が問い合わせ前に感じる不安
- AI導入支援サービスを、初見で理解させる順番
- 連絡したくなるサブページ構成
- そのままLP・サブページに使えるコピー素材

## ファイル一覧

- `01_Web集客成功サイトの共通パターン.md`
  - SmartHR、freee、カオナビ、Airレジ、Unbounce、NN/g、HubSpotを参考にした集客構造。
- `02_参考サイト分析.md`
  - 各サイトが、初見客に何を見せて信頼と行動を作っているか。
- `03_新規訪問者向けの訴求再設計.md`
  - 現LPを「知り合い向け」から「新規客向け」に変えるための設計。
- `04_サブページ企画.md`
  - 事例、料金納得、不安解消、無料診断ページの具体案。
- `05_問い合わせしたくなるコピー集.md`
  - ファーストビュー、CTA、問題提起、事例、FAQのコピー案。
- `06_実装優先順位.md`
  - どのページから作るべきか、既存LPのどこに導線を置くか。

## 重要な方針

新規客向けページでは、AIのすごさを説明しすぎない。

先に見せるべきなのは以下です。

1. 自分の会社の悩みだと分かる
2. 何をしてくれるのか一瞬で分かる
3. どんな成果物が残るのか分かる
4. 相談しても売り込まれすぎなさそうだと分かる
5. 誰がやるのか、信頼できる理由が分かる
6. 連絡後の流れが分かる

ゴールは「AIに興味がある人」ではなく、「社長依存・業務属人化・社員教育・確認作業に困っている中小企業」から問い合わせを取ること。
