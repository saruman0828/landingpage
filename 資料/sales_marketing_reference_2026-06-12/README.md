# 売れているセールスマーケティング実例リサーチ

作成日: 2026-06-12

対象: 中小企業経営者向け AI（Artificial Intelligence、人工知能）導入支援 LP（Landing Page、広告などから着地する販売ページ）

## このフォルダの中身

- `reference_report.html`  
  見やすく読むためのHTML資料です。ブラウザで開く用。
- `01_examples.md`  
  参考にした実例10件の要点、使える型、LPへの転用案。
- `02_lp_action_plan.md`  
  今のLPに何を足すべきかの優先順位。
- `03_copy_bank.md`  
  そのまま叩き台にできる見出し、CTA（Call To Action、行動喚起ボタン）、事例見出し案。
- `sources.csv`  
  参照元URL一覧。

## 結論

今のLPはファーストビューの訴求は強いですが、まだ「証拠」「業種別の自分ごと化」「無料相談前の低ハードル資料」が弱いです。

優先して入れるべきは次の3つです。

1. 業種別の雑務削減例
2. 無料相談の前に置く「3分診断」または「雑務AI化チェックリスト」
3. 導入後の具体的な変化をBefore/Afterで見せる事例枠

無理に大企業風のAI表現へ寄せるより、「社長が毎日抱えている確認・連絡・転記を減らす」という現場寄りの言葉を磨いた方が、この商材には合います。
